By default images are responsive and will scale up to their maximum width.

By default SVG graphics are responsive. For information on browser support visit Can I use.

Responsive Images Using Picturefill

To take advantage of the forthcoming W3C HTML standard <picture> element the styleguide also includes support for Picturefill. Picturefill is a Javascript polyfill that enables support for the <picture> elements in browsers that don’t support it natively. Support is also added for <img> with srcset and sizes.
