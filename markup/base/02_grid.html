Offset Columns

Columns can be offset to create whitespace. By default a column is offset on the left by the number of columns specified. So in a 12 column grid where we want an 8 column block to be centered we can offset it by 2 ((12 - 8)/2 = 2) to create the same amount of whitespace on each side.
